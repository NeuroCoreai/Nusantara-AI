# Nusantara-AI
AI Nusantara adalah website interaktif yang menyinergikan kecerdasan artifisial dengan pelestarian budaya Jawa Timur dan energi terbarukan dalam platform edukasi responsif untuk semua perangkat, menampilkan eksplorasi budaya, demo AI, peta energi, kalkulator hijau, dan proyek inovatif siswa.
